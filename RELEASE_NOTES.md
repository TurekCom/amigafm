# Amiga FM 0.1.1

Wydanie poprawkowe po pierwszym prototypie.

## Zmiany

- Panele przy pierwszym uruchomieniu pokazują listę dysków.
- Program zapamiętuje ostatnią lokalizację lewego i prawego panelu.
- Jeśli zapisana lokalizacja panelu nie jest dostępna przy uruchomieniu, panel automatycznie pokazuje listę dysków.
- Instalator instaluje program w `Program Files`, więc wymaga uprawnień administratora.
- Instalator nadal tworzy skrót w menu Start i ikonę na pulpicie.

## Pobieranie

- `AmigaFM-Setup-0.1.1.exe` - instalator Windows x64.

## Wymagania

- Windows 10 lub nowszy, 64-bit.
- 7-Zip zainstalowany osobno do operacji na archiwach.
- NVDA opcjonalnie, jeśli użytkownik chce korzystać z komunikatów czytnika ekranu.
