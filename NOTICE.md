# Notices

## NVDA Controller Client

Amiga FM distributes `nvdaControllerClient.dll` so the application can send speech messages to NVDA.

The NVDA Controller Client API is licensed under the GNU Lesser General Public License, version 2.1. The license text is included in `license.txt`.

## 7-Zip

Amiga FM starts the user's installed `7z.exe` for archive and disk image operations. 7-Zip is not bundled with the installer.

If archive support is needed, install 7-Zip separately from the official project distribution or through a trusted Windows package manager.
